Программно-аппаратный комплекс
видеофиксации и верификации
целостности продукции (ПАК )

Промышленное решение, которое фиксирует каждую банку в момент укупорки и формирует неоспоримую цифровую доказательную базу для арбитража с маркетплейсами. Снижение фиктивных возвратов до 85% за первый год эксплуатации.

01 · Бизнес-контекст

Подмена продукции на пути к покупателю

На маркетплейсах фиксируется устойчивая схема: банку вскрывают на транспортном плече или в ПВЗ, оригинальную краску сливают, заливают суррогат и возвращают как «брак». Без инструментального контроля на конвейере доказать подмену в арбитраже невозможно.
45%
доля фрода с возвратами

В структуре нецелевых финансовых потерь селлеров e-commerce махинации с подменой занимают лидирующее положение.

3 этапа
где возможна скрытая подмена

Транспортировка, складской транзит, пункт выдачи заказов. На любом из них третьи лица могут вскрыть тару, слить оригинал и залить суррогат — без видимых следов снаружи.

−85%
прогноз снижения потерь

Прогноз департамента бизнес-аналитики Neat Studio после внедрения ПАК  в течение года эксплуатации.

Цель проекта

На финишном участке конвейера (зона заполнения тары и закрытия крышки) — автоматизированная фиксация каждой банки. Видеоархив доказывает, что в момент выхода с завода продукт обладал герметичностью, целостностью пломб и оригинальным содержимым.

02 · Архитектура решения

Два сценария привязки видео
к маркировочным данным

Инженерный департамент Neat Studio выделил два сценария логической привязки. Рекомендуется Вариант 2: повышенная нагрузка компенсируется сквозной автоматизацией работы службы безопасности.
01
Вариант 1
Попартийная привязка

Непрерывная запись общего видеофайла на всю технологическую партию (например, 10 шт.). Распознанные коды связываются в БД с ID партии и временным интервалом.

Нагрузка
Минимальная: низкая частота транзакций, малые накладные расходы
Арбитраж
Низкая эффективность: ручной поиск фрагментов в длинных логах
Стоимость
410 000 ₽ · 3 недели
★ Рекомендуем
02
Вариант 2
Поштучная индексация

Алгоритмы Object Tracking. Для каждой банки формируется изолированный видеофрагмент, жёстко привязанный к её уникальному QR-коду. Выгрузка нужного ролика «сканирование — укупорка» в один клик.

Нагрузка
Выше средней: постоянная сегментация медиапотока в реальном времени
Арбитраж
Максимальная: 5–10-секундный ролик по любому QR-коду
Стоимость
от 680 000 ₽ · 5–8 недель
03 · Состав комплекса

Аппаратное обеспечение
и программный стек

Промышленный класс оборудования и модульный асинхронный стек по стандартам системной гигиены Neat Studio. Расчёт латентности — ≤ 80 мс на кадр на любом тарифе.

Аппаратный комплекс

MV-камера
Global Shutter · GigE Vision

30–60 FPS, GPIO-триггер. Сенсор экспонирует все пиксели одновременно — исключает «эффект желе» на движущемся конвейере.

LED-освещение
Импульсный «свет под камеру»

Стабильная контрастность зоны маркировки. Устраняет паразитные блики от металлических и глянцевых элементов крышки.

Edge PC
Intel i5/i7 · 16 ГБ RAM

NVMe SSD для буферизации медиапотока. ОС Linux (Ubuntu Server LTS) для максимального аптайма.

Кабельные трассы
STP Cat6 · до 100 м

Несжатый видеопоток по экранированной витой паре. Сервер выносится из зоны вибраций в защищённое помещение.

Виброгашение
Регулируемый кронштейн

Индустриальное надконвейерное крепление. Юстировка оптической оси под скорость и высоту банки.

NAS-архив · Корпоративный
RAID 4×4 ТБ

Отказоустойчивый дисковый массив. Срок хранения видеоархива — не менее 60 календарных дней.

Программный стек

Computer Vision Core

Python + GStreamer для низкоуровневых медиа-конвейеров реального времени. Аппаратное сжатие H.264/H.265 и упаковка сегментов через FFmpeg.

Каскадный ансамбль декодеров

Параллельная обработка кадра тремя независимыми декодерами: pyzbar, ZXing, OpenCV QRCodeDetector. Цель — 100% точность в условиях промышленного загрязнения.

Core Backend · FastAPI

Асинхронный оркестратор: обработка аппаратных событий, сопоставление маркировок с метаданными, управление путями сохранения. Авторизация OAuth2 + JWT.

PostgreSQL + SHA-256

Реляционная СУБД: ID партий, timestamps, связи с видеоархивом. Контрольная сумма SHA-256 для каждого фрагмента — защита от незаметной подмены.

≤ 30 мс

Дорожная карта
на 8 недель

Регламент Neat Studio для флагманской сборки. Базовый и Стандарт укладываются в часть этапов и завершаются за 3 и 5 недель соответственно. От Заказчика — доступ к конвейеру, электропитание с ИБП, монтаж кронштейнов и прокладка кабельных трасс.

  • Разработка ПО: скрипты CV, бэкенд FastAPI, база данных, фронтенд (для Корпоративного)
